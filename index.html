Link Up

LinkUp is a social media platform where users can connect with others, share content, and participate in discussions. This project includes functionalities such as user registration, login, post creation, comments, likes, tags, user roles, and follower relationships.





Shrouded

This project implements a quest system for a Unity game. The system includes various components such as quest givers, quest assets, and dialogue nodes to manage and display quests within the game. Players can interact with characters to receive, progress, and complete quests. The system ensures quests are dynamically loaded and associated with the correct quest givers.

Christmas Class Animation

This project showcases a Christmas-themed animation, centered around a rubber hose animation style. It features key animation techniques such as puppet pins, keyframes, and classic animation principles. The scene includes animated elements like a tree, snowman, and a cabin, and concludes with a festive "Merry Christmas" message. This animation was showcased at Valletta during Christmas time.

Space Explorers

This is a hyper casual survival game called Space Explorers, made for PC. In this game, the player will control a rocket that is endlessly moving forwards to the right. The player must avoid all obstacles, such as planets to continue going. The aim of the game is to get the highest score you can by going as long as you can.
